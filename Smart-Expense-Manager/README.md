# Smart Expense Manager

## Objective

A Python and MySQL project to track and analyze daily expenses.

## Features

* Add User
* Add Expense
* View Expenses (JOIN)
* Filter by Category and Date
* Total Expense Calculation
* Category-wise Spending
* Monthly Report
* Highest Expense
* Smart Insights
* Update and Delete

## Concepts Used

* OOP (Encapsulation, Inheritance, Abstraction)
* Functional Programming (map, filter, reduce)
* SQL Joins

## How to Run

1. Run `db.sql` in MySQL
2. Update password in `main.py`
3. Install requirements:

```
pip install -r requirements.txt
```

4. Run:

```
python main.py
```
